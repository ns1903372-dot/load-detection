    Research Context

    The original repository contains preprocessing and EEG notebooks. This app turns that into a deployable product surface.

    What the backend models

    • EEG attention strain from focus index and theta/beta balance.
    • Physiological stress via HRV and ocular fatigue proxies.
    • Behavioral friction via typing accuracy, hesitation, and context switching.
    • Actionable recommendations for real-world monitoring dashboards.
